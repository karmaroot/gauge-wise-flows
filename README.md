# SGVI – Sistema de Gestión y Validación de Indicadores

**ÁREA DE GESTIÓN ESTRATÉGICA (AGE) - COMISIÓN NACIONAL DE RIEGO**

## Contexto del Proyecto

Sistema de gestión y validación de indicadores institucionales. Este proyecto permite el reporte, revisión y aprobación de indicadores de gestión interna de forma eficiente y segura.
