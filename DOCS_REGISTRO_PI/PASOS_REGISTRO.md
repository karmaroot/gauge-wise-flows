# Guía para el Registro de Propiedad Intelectual en Chile (DDI)

Para proteger el software **SGI (Sistema de Gestión de Indicadores)** bajo derecho de autor, se debe realizar el trámite ante el **Departamento de Derechos Intelectuales (DDI)**.

## 1. Identificación del Trámite
- **Institución**: Departamento de Derechos Intelectuales (DDI).
- **Categoría**: Registro de programas de computación (software).
- **Plataforma**: Portal **CRIN** (Centro de Registros Integrados Nacionales).
- **URL**: [https://www.propiedadintelectual.gob.cl/](https://www.propiedadintelectual.gob.cl/)

## 2. Documentación a Presentar
En la plataforma CRIN se deberán cargar los siguientes documentos (los cuales se encuentran preparados en esta carpeta):

1.  **Código Fuente**: Una muestra representativa o el código íntegro (ver `CODIGO_FUENTE_MUESTRA.md`).
2.  **Manual de Usuario**: Descripción de las funciones del sistema (ver `MANUAL_USUARIO.md`).
3.  **Memoria Técnica**: Descripción técnica y arquitectura (ver `MEMORIA_TECNICA.md`).
4.  **Título/Nombre**: "SGI - Sistema de Gestión de Indicadores".
5.  **Autor y Titular**: Marcelo Silva Magna.

## 3. Pasos para el Registro
1.  **Ingreso**: Acceder al portal CRIN con Clave Única.
2.  **Formulario**: Completar los datos de la obra (Nombre: SGI, Género: Software/Programa de Computación).
3.  **Carga de Archivos**: Subir el archivo comprimido (ZIP) con toda la documentación. El límite es de 30MB para carga directa; si es mayor, se debe coordinar entrega física o vía enlace.
4.  **Pago de Aranceles**: Realizar el pago de la tasa de inscripción (aprox. 10% de una UTM o según valor vigente).
5.  **Resolución**: El DDI emitirá un número de registro y un certificado de inscripción que acredita la propiedad intelectual.

## 4. Recomendaciones Legales
- El registro otorga una presunción legal de autoría.
- Al ser una obra por cuenta propia (no por encargo laboral directo sobre el código), la titularidad le pertenece íntegramente al creador.
- Se recomienda realizar el registro antes de realizar despliegues masivos o entregar copias del código fuente a terceros.
