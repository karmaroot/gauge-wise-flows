# Manual de Usuario - SGI (Sistema de Gestión de Indicadores)

## 1. Introducción
El **Sistema de Gestión de Indicadores (SGI)** es una plataforma diseñada para centralizar, gestionar y validar indicadores críticos. Permite a los usuarios reportar avances, adjuntar evidencias y mantener un registro auditado de los ciclos de revisión.

## 2. Acceso al Sistema
- El acceso se realiza mediante correo electrónico y contraseña.
- El sistema cuenta con validación de sesiones segura y protección de datos a nivel de usuario.

## 3. Roles y Funcionalidades

### 3.1. Perfil Informante
- **Bandeja de Entrada**: Visualización de indicadores pendientes de reporte.
- **Reporte de Indicadores**: Ingreso de datos numéricos (numerador/denominador) y comentarios.
- **Gestión de Evidencias**: Carga obligatoria de documentos de respaldo en formato PDF, Excel o Imagen.
- **Correciones**: Capacidad de responder a observaciones técnicas de los revisores y reenviar reportes corregidos.

### 3.2. Perfil Revisor
- **Dashboard de Control**: Estadísticas generales de los reportes en curso.
- **Validación Técnica**: Revisión detallada de los datos y descarga de evidencias adjuntas.
- **Gestión de Observaciones**: Facultad para aprobar reportes o devolverlos con observaciones detalladas para su corrección.

### 3.3. Perfil Administrador
- **Gestión de Usuarios**: Creación y asignación de roles.
- **Configuración de Indicadores**: Definición de metas, periodicidad y fórmulas de cálculo.
- **Periodos de Reporte**: Apertura y cierre de ciclos de reportabilidad.

## 4. Flujo de Trabajo (Workflow)
1. **Inicio de Periodo**: El sistema habilita los indicadores según su periodicidad.
2. **Carga Informante**: El usuario completa los datos y sube la evidencia.
3. **Revisión Técnica**: El revisor analiza la información.
   - Si es **Correcto**: El sistema genera un registro de aprobación definitiva.
   - Si hay **Errores**: Se emite una observación y el reporte vuelve al informante.
4. **Respuesta**: El informante corrige los datos indicados y la evidencia, cerrando el ciclo.
